# Iodine value

$$IV = 0.6683 \times DU + 25.0364$$

## Abbreviations

- IV

## Description

Represents the iodine value (IV), which is an index of the number of double bonds in biodiesel. It serves as a key parameter that quantifies the degree of unsaturation (DU) of the fuel. A higher iodine value indicates a greater number of double bonds, which typically correlates with poorer oxidation stability but better cold flow properties.
