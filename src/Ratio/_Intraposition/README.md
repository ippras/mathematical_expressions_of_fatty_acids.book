# Fatty acids intraposition ratio

The ratio between fatty acids in the same stereospecific position.

- Fish lipid quality
  - FLQ
- Health-promoting index
  - HPI
- Hypocholesterolemic to hypercholesterolemic fatty acids
  - H/H
- Index of atherogenicity
  - IA
- Index of thrombogenicity
  - IT
- Linoleic fatty acid to α-Linolenic fatty acid
  - LA/ALA
- Polyunsaturated omega -6 to polyunsaturated omega -3 fatty acids
  - $PUFA(\omega-6)/PUFA(\omega-3)$
- Polyunsaturated to saturated fatty acids
  - PUFA/SFA
