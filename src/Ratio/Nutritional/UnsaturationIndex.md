# Unsaturation index

`nutritional`

$$\sum N \times \ce{NUFA}$$

## Abbreviations

- UI

## Description

Represents the unsaturation index, which is a measure of the number of unsaturated bonds in the fatty acid moieties of lipids (such as phospholipids). It is calculated as the fraction of each unsaturated fatty acid multiplied by the number of double bonds within that specific fatty acid. This index provides a comprehensive evaluation of the overall degree of unsaturation in a lipid sample, which is relevant for assessing membrane fluidity, oxidative stability, and nutritional quality.
